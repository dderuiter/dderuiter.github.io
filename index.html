C&C

Login Page
Description:

The C&C web application aides law enforcement during an active shooter event.

The back-end for the project is written in Java using the Spring MVC framework. Pages are rendered using JavaServer Pages (JSP).

The database for the application is MySQL and data is requested from the database using the JDBC API. Queries are written using SQL.

The front-end code utilizes Javascript and jQuery to dynamically manipulate the DOM. Data is requested from the front-end using AJAX, which submits asynchronous requests to a custom built API that pulls data from the database. After a successful request, the data is presented using a combination of Bootstrap and custom CSS.

QMBES

Home Tab - Before Run

View on GitHub

Description:

This project is an implementation in Java of the mathematical Quine McCluskey algorithm for minimizing Boolean expressions.

Also, it implements Petrick’s method for deducing all minimum sum-of-products solutions from a prime implicant chart after using the Quine McCluskey method to form a prime implicant chart.

The front-end was developed with JavaFX (the latest GUI library for Java) using Gluon's Scene Builder application for the basics.

NSA Codebreaker Challenge


Overview

The challenge provides students with a hands-on opportunity to develop their reverse-engineering / low-level code analysis skills while working on a realistic problem set centered around the NSA's mission.


Details

2016

The fictional backstory for the challenge is as follows: Terrorists have recently developed a new type of remotely controlled Improvised Explosive Device (IED), making it harder for the U.S. Armed Forces to detect and ultimately prevent roadside bomb attacks against troops deployed overseas. The National Security Agency (NSA), in accordance with its support to military operations mission, has been asked to develop capabilities for use against this new threat. This will consist of six tasks of increasing difficulty, with the ultimate goals of being able to disarm the IEDs remotely and permanently render them inoperable without the risk of civilian casualties.


Results

  • Tasks 1-4: Completed

Tasks

  • Task 1: Compute a hash and identify IED network ports (Completed)
  • Task 2: Refine IED network traffic signature (Completed)
  • Task 3: Decrypt IED key file (Completed)
  • Task 4: Disarm an IED with the key (Completed)
  • Task 5: Disarm any IED without a key
  • Task 6: Permanently disable any IED
